# 🛒 Amazon E-Commerce Business Intelligence Dashboard

## 📌 Project Overview

This project analyzes Amazon E-Commerce sales data using **Python, Pandas, NumPy, and Matplotlib** to uncover meaningful business insights. The analysis focuses on sales performance, customer purchasing patterns, regional trends, and operational metrics through data cleaning, feature engineering, KPI analysis, and data visualization.

The goal of this project is to demonstrate practical data analysis skills by transforming raw sales data into actionable business insights.

---

## 🎯 Objectives

* Clean and preprocess real-world sales data.
* Perform exploratory data analysis (EDA).
* Calculate key business performance indicators (KPIs).
* Visualize sales trends and customer behavior.
* Generate business insights and recommendations.

---

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

---

## 📂 Dataset

The dataset contains Amazon sales transactions, including:

* Order ID
* Date
* Order Status
* Product Category
* Quantity
* Revenue
* Customer Location
* Shipping Details
* Promotions
* B2B/B2C Orders

Dataset Source: Kaggle

---

## 📊 Key Performance Indicators (KPIs)

* Total Revenue
* Total Orders
* Total Quantity Sold
* Average Order Value (AOV)

---

## 📈 Visualizations

The project includes the following analyses:

* Monthly Revenue Analysis
* Revenue by Product Category
* Best Selling Product Sizes
* Order Status Distribution
* Top 10 States by Revenue
* B2B vs B2C Orders
* Promotion Analysis

## 💡 Key Insights

* Identified monthly revenue trends.
* Analyzed the highest revenue-generating product categories.
